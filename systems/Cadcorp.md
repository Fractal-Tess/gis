# Research "Cadcorp"

Cadcorp is a UK-based developer and supplier of Geographic Information System (GIS) and web mapping software[^5]. The Cadcorp SIS (Spatial Information System) product suite is an integrated family of web, server, desktop, and developer GIS products that run on Microsoft Windows[^3]. Cadcorp is an ISO 9001:2000 and ISO/IEC 27001:2005 certified company, a Microsoft SQL Server Spatial Partner, and an Ordnance Survey Licensed Developer Partner[^2].

**Core Functionalities:**

- **Cadcorp SIS WebMap:** An off-the-shelf GIS application for internet and intranet web mapping that presents and allows users to query GIS spatial data. It has a mobile responsive web mapping interface[^1].
- **Cadcorp SIS Desktop:** Provides a full range of GIS functionality, including data creation, querying, analysis, and modeling[^6]. It can perform advanced 3D spatial analysis on a variety of spatial data formats[^4].
- **GeognoSIS:** A web mapping and geospatial web server used in the development and deployment of web applications that use maps and other geographic information[^3].
- **Map Express:** A free-to-use desktop GIS product that supports various data formats. It allows users to perform spatial analysis and visualize data in 3D[^5].

**Ease of Use and Learning Curve:**

- Cadcorp SIS products utilize common GIS algorithms, APIs, and data source methodologies across all operating environments[^3].
- The desktop version has a Microsoft Office Fluent user interface[^5].
- An intuitive ribbon interface allows access from multiple devices[^6].

**Data Formats Supported:**

- Cadcorp SIS supports over 160 GIS, CAD, raster, and database formats[^3].
- It supports Ordnance Survey formats[^5].
- It can perform advanced 3D spatial analysis on over 300 different formats of spatial data[^4].

**Scalability and Performance:**

- Cadcorp provides dedicated resources for each customer[^1].
- A new proprietary data format offers access to vector, raster, and topological data while reducing memory requirements[^5].
- Performance improvements have been made, and desktop and SIS Control 3D windows have been rewritten to take advantage of modern CPU and GPU hardware[^5].

**Integration Capabilities:**

- Cadcorp SIS links directly to the Ordnance Survey Data Hub and has dedicated guides for linking to the OS Features API, the OS Maps API, and the OS Vector Tile API[^2].
- It offers direct connections to databases (Oracle, SQL Server, and PostGIS) without the use of middleware[^3].
- Cadcorp has been a member of the Open Geospatial Consortium (OGC) since 1997 and has multiple OGC-compliant products[^2].

**Pricing and Licensing:**

- Cadcorp SIS WebMap is priced at £1,250 per virtual machine per month[^1].
- Cadcorp SIS Desktop is priced at £2,600 per user per year[^6].
- A free trial may be available[^6].

**Community and Support:**

- Online documentation is available on the Cadcorp website[^1].
- Onsite or online training is provided as an option[^1].
- All incidents should be reported to the Cadcorp support desk[^1].
